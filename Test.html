I. Cours WEB

Différence entre web et internet : ce n’est pas la même chose, le web fais partie d’internet, car dans internet il y a : le web, les messageries instantanées, les e-mails. Le web est une infime partie d’internet. Un site web est un ensemble de ressources ou bien de pages qui vont avoir une logique les unes avec les autres. HTML permet de faire du contenu : HyperText Markup Language C’est un langage de balise (prédéfini) tout le mode se sert des mêmes. A la différence du XML qui lui a des balises libres. Le HTML en est à sa version 5 CSS : Cascading Style Sheet Le css est utilisé pour « faire du beau » Un navigateur permet de consulter les sites web (captain obvious) Le langage HTML a été inventé par un certain Tim Berners-Lee en 1991 est l’inventeur du web HTML5 est version la plus stable elle permet d’inclure des vidéos des fonctionnalités utilise pour gérer des formulaires En CSS nous somme au CSS3 c’est grâce au css3 que la cadre arrondie existe les dégradés des ombres etc… C’est grâce css3. L’architecture du web est une architecture client-serveur. Lorsque l’on se connecte à une url on demande à l’hébergeur les ressources pour la page à afficher. En local le protocole n’est pas https mais File ça simule le client-serveur. Selon le navigateur le même code peut ne pas s’afficher à 100% pareille selon le navigateur Le cache est une zone mémoire alloué aux navigateurs. La mémoire cache rend l’obtention d’infos plus rapide en la cherchant en local.
